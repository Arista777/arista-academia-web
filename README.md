# Arista Martial Arts - Sitio Web (React)

Proyecto configurado para funcionar de dos formas:

## Opcion 1: VS Code + Go Live (la mas simple)

1. Abre la carpeta:
   - `/Users/gato_de_monte/Documents/New project/arista-academia`
2. En VS Code, abre `index.html`.
3. Pulsa `Go Live`.

Con esto ya se debe ver la pagina.

## Opcion 2: React + Vite

1. Instala dependencias:
   - `npm install`
2. Levanta servidor:
   - `npm run dev`

## Donde editar contenido

Si usas `Go Live`, edita:

- `src/live-data.js`
- `public/arista-logo.svg` (puedes reemplazarlo por tu logo final)

Si usas Vite, edita:

- `src/data.js`
